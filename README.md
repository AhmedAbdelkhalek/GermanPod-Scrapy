# GermanPod-Scrapy

This is a simple spider to download free 100 words data from Germanpod101.com 
https://www.germanpod101.com/german-word-lists/

# Downloaded items 
- csv contain words
- images 
- german sound files
- english sound files


# Usage
- scrapy crawl wordsSpider -o words.csv 
- Or just run main.py using cmd or any IDE
